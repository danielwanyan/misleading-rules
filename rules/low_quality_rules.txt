# 视频低质问题判定规则

## 版本信息

- 当前版本：2025-05-19-v1
- 上次更新：2025-05-19

---

## 规则索引

| 编号 | 规则名称 | 说明 |
|------|---------|------|
| LQ01 | AIGC纯商品展示 | AIGC + 纯商品/无人/纯肢体/捂嘴流 |
| LQ02 | AIGC仅营销叫卖 | AIGC + 无商品信息 + 只有营销话术 |
| LQ03 | 使用重复音频 | 30日内同语义音频 > 30 |
| LQ04 | 内容介绍不详细 | 电子/护肤/保健品必须展示+说明 |
| LQ05 | 非AIGC仅营销叫卖 | 非AIGC + 只有营销话术 |

---

## 输入信息

### 文本信息

| 字段 | 说明 |
|------|------|
| ASR | 视频语音转文本 |
| OCR | 视频画面中识别出的文字 |
| Comment | 视频评论/描述 |
| Product_title | 产品标题 |
| Product_category | 产品分类 |

### 图片信息（视觉理解）

| 字段 | 说明 |
|------|------|
| URL_Frames | 视频帧图片URL（多张） |
| Product_image | 产品主图URL |

### 外部数据打标

| 字段 | 说明 |
|------|------|
| is_aigc | 是否为AIGC生成（true/false） |
| audio_duplicate_count | 30日内同语义音频出现次数 |

---

## 判定流程

1. 检查外部数据打标（is_aigc, audio_duplicate_count）
2. 逐条检查 LQ01-LQ05 规则
3. 任何一条规则命中 → 判定为低质
4. 检查是否适用 Exception（豁免）
5. 如适用豁免 → 改判为非低质

---

## 规则详情

### LQ01：AIGC纯商品展示

**触发条件**（必须同时满足）：
1. `is_aigc = true`（外部打标）
2. 视频内容属于以下任一情况：
   - 纯商品展示（无人物出现）
   - 只有纯肢体展示（如只有脚、手等局部）
   - 捂嘴流（主播捂嘴，只展示商品不说话）

**判定逻辑**：
- AIGC 标记为 true，且视频中没有真人主播进行正常介绍 → 命中

---

### LQ02：AIGC仅营销叫卖

**触发条件**（必须同时满足）：
1. `is_aigc = true`（外部打标）
2. 完全无商品信息介绍
3. 只有营销叫卖内容

**营销叫卖关键词**：
- 价格：XX元、只要XX、特价、优惠价
- 促销：买一送一、买赠、限时优惠、活动今天截止
- 包邮：包邮、免运费、free shipping
- 断货：即将断货、库存紧张、最后XX件
- 其他：错过再等一年、不买后悔

**判定逻辑**：
- AIGC + 只有营销话术 + 无任何商品功能/特点介绍 → 命中

---

### LQ03：使用重复音频

**触发条件**：
- `audio_duplicate_count > 30`（外部打标）

**判定逻辑**：
- 30日内同语义音频出现超过30次 → 命中

---

### LQ04：内容介绍不详细

**核心规则**：

| 商品类别 | 要求 |
|---------|------|
| 电子/数码产品 | 必须展示 + 必须说明 |
| 护肤品 | 必须展示 + 必须说明（卸妆水除外） |
| 保健品 | 必须展示 + 必须说明 |
| 其他商品 | 只要展示完整清楚即可 |

**各类别定义**：

#### 1. 电子/数码产品（必须展示+说明）

**包含**：
- 智能手机、平板电脑、笔记本电脑
- 游戏机（PS5、Switch等）
- 插电使用的电器（有使用成本的设备）
- 耳机、音箱等数码配件

**要求**：
- 必须展示产品外观/功能
- 必须有口头或文字说明产品特点

#### 2. 护肤品（必须展示+说明）

**包含**：
- 爽肤水（toner）
- 精华（essence/serum）
- 面霜（facial cream）
- 眼霜（eye cream）
- 其他功效型护肤产品

**豁免**：
- 卸妆水（makeup remover）→ 不需要说明

**要求**：
- 必须展示产品
- 必须有功效介绍

#### 3. 保健品（必须展示+说明）

**包含**：
- 维生素（vitamins）
- 蛋白粉（protein）
- 膳食补充剂
- 其他健康补充产品

**要求**：
- 必须展示产品
- 必须有成分/功效说明

#### 4. 其他商品（展示清楚即可）

**包含**：
- 服装、配饰
- 家居用品
- 化妆品（彩妆类）
- 其他非上述类别的商品

**要求**：
- 只要在视频中清晰、完整地展示即可
- 不需要口头说明

---

### LQ05：非AIGC仅营销叫卖

**触发条件**（必须同时满足）：
1. `is_aigc = false`（外部打标）
2. 视频口播内容只有营销叫卖
3. 没有任何商品介绍内容

**营销叫卖关键词**（同 LQ02）：
- 价格类：XX元、只要XX、特价
- 促销类：买一送一、限时优惠、活动截止
- 包邮类：包邮、免运费
- 稀缺类：即将断货、库存紧张

**判定逻辑**：
- 非AIGC + 只有营销话术 + 无任何商品功能/特点介绍 → 命中
- 不区分类目，所有类目都适用

---

## Exception（豁免情况）

以下情况即使命中上述规则，也判定为 **非低质（OK）**：

### Exception 1：化妆视频（Makeup Video）

**场景**："Get Ready With Me"（GRWM）类型

**特征**：
- 主播简要介绍（或完全不提）绑定的化妆产品
- 然后演示上妆过程和效果

**判定为 OK 的原因**：
1. 清晰展示了产品
2. 展示了主播推广产品的意图
3. 让观众能够评估产品

**注意**：
- 这是一个连续的动作
- 即使绑定的产品没有在整个 GRWM 过程中都被演示，也算 OK

---

### Exception 2：卡牌开箱视频（Card Unboxing Video）

**场景**：卡牌拆箱/开包直播

**特征**：
- 展示可开的卡包/卡牌类型
- 倒计时等待开箱
- 与观众互动（讨论队伍、角色等）
- 开卡过程

**特殊性**：
- 卡牌内容在开包前无法展示（类似书籍，剧透会失去意义）
- 信息有限是这类内容的固有属性

**判定为 OK 的原因**：
1. 清晰展示了绑定的卡牌产品
2. 主播在推广即将到来的开箱
3. 在谈论卡牌本身
4. 与观众互动
5. 展示开箱已购买的卡牌

---

### Exception 3：派对桌游介绍视频（Party Board Game Intro Video）

**场景**：桌游介绍/演示

**判定为 OK 的条件**：
- 展示了游戏玩法过程

---

## 冲突处理规则

1. 先检查是否命中 LQ01-LQ05
2. 如命中，再检查是否适用 Exception
3. 如适用 Exception → 改判为 OK
4. 如不适用 Exception → 保持低质判定
